## Описание проекта

Данный проект представляет собой консольную систему системы учета книг, читателей и выдачи книг. А также ранение информации о книгах, читателях и историях выдачи.

---

## Функционал программы

Программа позволяет:

- добавлять книги в библиотеку;
- добавлять читателей;
- просматривать список книг;
- просматривать список читателей;
- выдавать книги читателям;
- возвращать книги;
- сохранять данные в файл (JSON);
- загружать данные из файла.

---

## Используемые технологии

- Python 3
- ООП (классы и объекты)
- Работа с файлами
- Формат JSON
- Консольный интерфейс

---

## Структура проекта

Проект состоит из следующих классов:

### Book
Хранит информацию о книге:
- название
- автор
- статус (выдана / свободна)

### Reader
Хранит информацию о читателе:
- имя
- список взятых книг

### Library
Основной класс системы:
- управление книгами
- управление читателями
- выдача и возврат книг
- сохранение и загрузка данных

---

## Хранение данных

Все данные сохраняются в файл:
       library.json

Файл содержит:
- список книг
- список читателей
- информацию о выданных книгах

---

## Запуск проекта

1. Открыть файл `library.py`
2. Запустить программу
3. Использовать консольное меню
